一个比robosuite更简洁的物理平台，专为通用机器人/人工智能/物理AI应用而设计。

教程

install

  1. 需求：
    1. Python: 3.9+ 所以我conda create -n genesis python==3.10
    2. OS: Linux (recommended) / MacOS / Windows 但windows不太方便。
  2. pip install genesis-world
  3. 安装pytorch
  4. 以上就是最基础的安装，还有一些额外的根据需求安装即可
