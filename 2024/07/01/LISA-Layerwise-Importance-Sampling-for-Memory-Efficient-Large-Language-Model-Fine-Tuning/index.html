LISA: Layerwise Importance Sampling for Memory-Efficient Large Language Model Fine-Tuning

发表于 2024-07-01 分类于 paper

百篇paper计划(6/100)，好像是LoRA的改进 参考阅读https://www.jiqizhixin.com/articles/2024-04-01-13 - 论文标题：LISA: Layerwise Importance Sampling for Memory-Efficient Large Language Model Fine-Tuning - code：github - rank: - 打标签：

abstract

  1. 问题
    1. LLM消耗大量存储
    2. 例如，一个7B模型通常需要至少60 GB的GPU内存来进行全参数训练
  2. 同行
    1. 低秩自适应( Low-Rank Adapting，LoRA )等参数高效微调技术被提出以缓解这一问题。
    2. 然而，在大多数大规模微调设置中，它们的性能并没有达到完全参数训练的水平，因为它们将参数搜索限制在低秩子空间中。
  3. 理论研究
    1. 为了弥补这一不足（LoRA不好）
    2. 我们研究了LoRA在微调任务上的逐层性质(layerwise properties)
    3. 并观察到不同层之间权重范数的意外但一致的偏度。
  4. 应用理论研究提出的训练策略
