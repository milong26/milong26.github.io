codelab

本科学移动智能开发的时候用codelab学的，当时的作业也有要求，但是我比较懒所以请同学帮忙提交的，现在重新学一下。 官网 参考文档 遇事不决走官网，华为也有一个可以参考，本文所学习的主要是谷歌的。 但是为什么官网没有教怎么写codelabs，如果有的话记得删掉这句 配置 go-language Node.js-v10+以及npm claat 写 先找到准备存放codelabs项目的文件夹 下载工具(?个人理解是类似hexo搞的环境依赖) git clone https://github.com/googlecodelabs/tools 慢的话试试镜像 git clone https://github.com.cnpmjs.org/googlecodelabs/tools 在该文件夹里信件md文件即可写了，markdown语法都一样 开头可以有几个配置 summary: 测试，第一个codelab id: first_codelab categories: Android tags: android status: Published authors: kk Feedback Link: https://github.com/wkk-knight/ 如果在Windows系统写这个md 文件，使用的编辑器一定要用unix 的换行格式 ，否则你会发现输出的格式会有异常。这个换行符在vs code 等编辑器可以修改的。 执行命令 claat export FileName.md html文件可以看了，后面部署以后有空再写，我懒=)
