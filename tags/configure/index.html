配置java

搞开发（特别是软件比较多的）还得是看java啊 下载 官网 jdk和jre都是啥 JRE是java运行时环境而JDK是java开发工具包，JDK包含JRE，但是JRE可以独立安装。JDK 是用于开发 Java 程序的最小环境，JRE是提供给 Java 程序运行的最小环境。运行java程序只需安装JRE。如果需要编写java程序，需要安装JDK。 配置 cmd -> java -verbose 查看安装目录 这个就没有python智能了需要手动配置环境 在系统变量里面点新建 其他问题 带有相对地址的不要放在环境变量最开始，要不然环境变量就只显示一条横的很难改
