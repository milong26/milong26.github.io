pytorch之hook

hook，钩子。一般来说，“hook”是在特定事件之后自动执行的函数。 参考pytorch-hook文档 前情提要 x = torch.tensor([2.], requires_grad=True) a=x**2 b=a**3 b.backward() # 求grad和back是在一起的，算出来就是db/dx print(x.grad) 定义 PyTorch提供四种Hook函数： 1、torch.Tensor.register_hook(hook) 2、torch.nn.Module.register_forward_hook 3、torch.nn.Module.register_forward_pre_hook 4、torch.nn.Module.register_backward_hook torch.Tensor.register_hook 功能：注册一个反向传播hook函数，Hook函数仅一个输入参数，为张量的梯度。Hook不应修改其参数梯度值，但可以选择返回一个新的梯度，该梯度将代替grad使用。 hook(grad) -> Tensor or None def hook_grad(grad): print(grad) x = torch.tensor([1.], requires_grad=True) a=x*2 b=a**2 c=b**3 handle = b.register_hook(hook_grad) c.backward() print("x.grad:",x.grad) 在hook中输出的grad为dc/dc(backward/register) 最后输出的x.grad为dc/dx(backward/requires_grad=True) 更多内容看autograd torch.nn.Module.register_forward_hook 功能：注册module的前向传播Hook函数 参数： module：当前网络层 input：当前网络层输入数据 output：当前网络层输出数据 import torch import torch.nn as nn # 构建网络，一个卷积层一个池化层 class Net(nn.Module): def __init__(self): super(Net,self).__init__() self.conv1 = nn.Conv2d(1, 2, 3) self.pool1 = nn.MaxPool2d(2) def forward(self, x): # 卷积 x = self.conv1(x) # 池化 x = self.pool1(x) return x # 初始化网络 net = Net() # detach将张量分离 # fill_将tensor中的所有值都填充为指定的value # conv1的weight有2张，都是3x3，第一张全填充1，第二张全填充2 net.conv1.weight[0].detach().fill_(1) net.conv1.weight[1].detach().fill_(2) net.conv1.bias.detach().zero_() print(net.conv1) print(net.conv1.weight) print(net.conv1.bias) # fake_img bacth_size=1,input_channel=1,为4x4大小，每个格子都是1的二维 fake_img = torch.ones(1,1,4,4) # 定义forward_hook # input：当前网络层输入数据 # output：当前网络层输出数据 def forward_hook(module, data_input, data_output): print("output",data_output) print("input",data_input) # 注册Hook net.conv1.register_forward_hook(forward_hook) output = net(fake_img) 最后显示的是卷积层进行了一次卷积的输入和输出。 ...
