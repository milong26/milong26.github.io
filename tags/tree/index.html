树的计数相关问题

光看名字就比较高级了:D 学习这个的动力来自于一个问题：对于包含n个顶点的无向完全图最多包含$n^{n-2}$棵生成树。 Prüfer序列 Prüfer 序列可以将一个带标号 n 个结点的树用 [1,n] 中的 n-2 个整数表示。你也可以把它理解为完全图的生成树与数列之间的双射。常用于组合计数问题中。 对树建立序列 对树建立Prüfer序列的步骤：每次选择一个编号最小的叶结点并删掉它，然后在序列中记录下它连接到的那个结点。重复 n-2 次后就只剩下两个结点，算法结束。 先删除节点再记录标号，重复n-2之后只剩下根节点没删。 堆构造 听起来挺像建堆的过程，用堆的方法时间复杂度O(NlogN) 代码 // 结点是从 0 标号的 vector<vector<int>> adj;// adj的构造应该是：一行上每一个数表示和这行对应的点连接的节点序号 vector<int> pruefer_code() { int n = adj.size(); set<int> leafs; vector<int> degree(n); vector<bool> killed(n, false);//killed表示已经删除的 for (int i = 0; i < n; i++) { degree[i] = adj[i].size();//求每个点的度 if (degree[i] == 1) leafs.insert(i);//如果这行只有自己一个节点，就是度为0（degree=1）=叶子节点 } //求Prüfer code vector<int> code(n - 2); for (int i = 0; i < n - 2; i++) { int leaf = *leafs.begin();//叶子节点 leafs.erase(leafs.begin());//每次都会删掉一个节点 killed[leaf] = true; int v; for (int u : adj[leaf])//找leaf的邻接节点 if (!killed[u]) v = u; code[i] = v;//记录leaf的邻接节点 if (--degree[v] == 1) leafs.insert(v);//重新更新叶子节点 } return code; } 代码读懂了：一句话：按从小到大的顺序删除叶子节点，因为是树，code记录的就是被删除的叶子节点的parent 但是我不知道它有什么用.-. ...

最小生成树

满足两点之间有且仅有一条简单路径的结构只有树 且这棵树包含 n 个节点。我们称这棵树为给定的图的生成树，其中总权值最小的生成树，我们称其为最小生成树。 最小生成树关键在避免环 生成树 定义 一个连通图的生成树是一个极小的连通子图，它包含图中全部的n个顶点，但只有构成一棵树的n-1条边。 性质 生成树的属性 一个连通图可以有多个生成树； 一个连通图的所有生成树都包含相同的顶点个数和边数； 生成树当中不存在环； 移除生成树中的任意一条边都会导致图的不连通， 生成树的边最少特性； 在生成树中添加一条边会构成环。 对于包含n个顶点的连通图，生成树包含n个顶点和n-1条边； 对于包含n个顶点的无向完全图最多包含$n^{n-2}$颗生成树。 证明$n^{n-2}$： 为什么是n^n-2），一篇里面塞不下:D 最小生成树 定义 所谓一个带权图的最小生成树，就是原图中边的权值最小的生成树 ，所谓最小是指边的权值之和小于或者等于其它生成树的边的权值之和。 求解 Kruskal 贪心找边，从小到大加入边 按边长度排序 从小到大，if加入e之后没有形成环，e加入G' 最后得到G‘ 问题是判断成环，见其它post Prim 加点，把点分成两坨A,B集合，A表示已经成了，B是待选的，要做的事从B里面挑一个点，满足它到A的点里面，有个边距离是最短的 （理解成长三角经济圈画好了，接下来先从周边里挑一个离长三角圈里面的城市最近的，这么想有点奇怪） 关键在怎么从待选里找距离最短的点 思路是用一个点个数大小的数组，没有直接连用∞，否则记录最小值 注意区分已访问过的点 这个为什么不会成环？把点分成两坨了，找的都是新点 示例与代码 测试数据 1 6 10 1 2 28 6 5 25 5 7 24 2 7 14 2 3 16 3 4 12 4 7 18 4 5 22 代码 使用Kruskal算法 #include<iostream> #include<algorithm> using namespace std; const int maxe = 100; int fa[maxe], n, m, ans = 0; struct edge { int u, v, w; }e[maxe]; bool cmp(edge a, edge b) { return a.w<b.w; } int get(int x)//用处是看它是否成环的，nb { if (fa[x] == x) return x;//x的根为它本身，x就是该集的根 return fa[x] = get(fa[x]);//路径压缩 } int main() { cin >> n >> m; for (int i = 1; i <= n; i++) fa[i] = i;//初始化 for (int i = 1; i <= m; i++) { cin >> e[i].u >> e[i].v >> e[i].w; } sort(e + 1, e + 1 + m, cmp);//升序排序 for (int i = 1; i <= m; i++) { int u = get(e[i].u);//找根 int v = get(e[i].v); if (u == v) continue;//continue 表示终止本次（本轮）循环，进入下一轮循环。判断它是否成环 fa[u] = v; cout << e[i].u << " " << e[i].v << endl; ans += e[i].w; } cout << ans << endl; return 0; }
