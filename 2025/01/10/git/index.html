在错误的git仓库里面进行了git remote add upstream https://github.com/xx/xx.git的操作之后怎么撤销？

你可以运行以下命令查看当前的远程仓库列表：git remote -v

从github上clone了一个项目，自己改了一点，现在想pull原始代码一些新东西，又想保留自己的修改，怎么做

### 1️⃣ 在 GitHub 上创建自己的仓库**

  • 登录 GitHub，点击 New Repository

  • 取一个名字，例如 lerobot-my-version

  • 不要勾选 “Initialize with README”，保持空仓库

假设你的仓库地址是：

https://github.com/你的用户名/lerobot-my-version.git

2️⃣ 将你的本地仓库关联到自己的远程仓库

git remote rename origin upstream # 先把原来的远程重命名为 upstream

git remote add origin https://github.com/你的用户名/lerobot-my-version.git`

此时：

  • upstream → 原作者仓库
  • origin → 你自己的仓库
