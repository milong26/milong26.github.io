发表于 2024-12-19 更新于 2025-07-09 分类于 it

install

linux

  1. 应用商店的不好用，去官网

windows

配置

  1. 修改语言
    1. ctrl+shift+p，输入Configure Display Language
    2. 选择语言（需要下载）后重新启动该软件

开发

远程开发

连接SSH

  1. 连接过程中发现you are about to connect to an os version ，只能降低vscode版本了1.85
  2. 关闭自动更新：文件-preference-setting-update mode改为none
  3. 安装 Remote-SSH 插件

便捷使用

  1. select-for-compare 对比代码
    1. 点一个，select for compare
    2. 点第二个，compare with selected
  2. 比较当前代码和github官方目前的相差多少
    1. 插件 Gitlens emm貌似也不是很好用
    2. 直接把官方的拉过来当一个branch，然后直接比较分支算了
      1. 拉官方
        1. git checkout -b lerobot_official
        2. git fetch upstream (之前添加过了，git remote add upstream)
      2. gitlens有个compare的功能，选中两个分支，挺好用的，就是文件都是read-only

branch

  1. 创建分支
  2. 删除分支
    1. 本地：git branch -d name-of-branch （没合并的话会报错，用-D可以强制删除）
    2. 远程：git push origin --delete name-of-branch
