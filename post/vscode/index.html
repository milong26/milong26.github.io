售后问题#

  1. Error loading webview: Error: Could not register service worker: InvalidStateError: Failed to register a ServiceWorker: The document is in an invalid state.. 1、关闭vscode 2、按WIN + R，输入cmd，打开终端，然后输入命令 code --no-sandbox 3、重启vscode，就可以正常使用了。

修改文件显示顺序 在setting里面搜索sort order,explore对应的就是vs工作区的文件显示，写blog最好还是把文件按修改时间排序比较好。

为什么我的vscode不能点单词跳转到函数定义？ 要安装python插件，然后ctrl shift p输入 Python: Select Interpreter
