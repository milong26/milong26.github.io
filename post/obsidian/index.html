新建文件自动套用模板#

需求：写[[]]自动套用指定的模板：使用templater按路径指定模板就行，新文件创建时会自动应用 安装模板templater，它的https://silentvoid13.github.io/Templater/

第一步：设置模板文件夹 进入 Settings → Templater，找到 Template folder location，填写你存放模板的文件夹名：

template

（也就是你的 template/log.md 所在的那个 template 文件夹） 第二步：开启新文件触发 在同一个 Templater 设置页面，找到并开启：✅ Trigger Templater on new file creation 开启后，下方会出现新的配置项。

第三步：配置 Folder Templates（核心）

开启上面的开关后，会出现 Template matching mode 选项，选择： Folder templates 然后点击 Add rule，填写：

字段 填写内容
Folder /
Template template/log.md
Folder 填 / 表示全局兜底规则，所有文件夹下新建的文件都会套用这个模板。

第四步：排除模板文件夹（推荐） 为了防止 Templater 对模板文件夹内的文件也触发（比如你新建 template/xxx.md），可以在 Excluded folders 里添加：

template

来检验一下效果，我现在在md文件里面写[[git设计理念]]，成功！
