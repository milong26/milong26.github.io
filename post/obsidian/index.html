新建文件自动套用模板#

需求：写[[]]自动套用指定的模板：使用templater按路径指定模板就行，新文件创建时会自动应用 安装模板templater，它的https://silentvoid13.github.io/Templater/

第一步：设置模板文件夹 进入 Settings → Templater，找到 Template folder location，填写你存放模板的文件夹名：

template

（也就是你的 template/log.md 所在的那个 template 文件夹） 第二步：开启新文件触发 在同一个 Templater 设置页面，找到并开启：✅ Trigger Templater on new file creation 开启后，下方会出现新的配置项。

第三步：配置 Folder Templates（核心）

开启上面的开关后，会出现 Template matching mode 选项，选择： Folder templates 然后点击 Add rule，填写：

字段 填写内容
Folder /
Template template/log.md
Folder 填 / 表示全局兜底规则，所有文件夹下新建的文件都会套用这个模板。

第四步：排除模板文件夹（推荐） 为了防止 Templater 对模板文件夹内的文件也触发（比如你新建 template/xxx.md），可以在 Excluded folders 里添加：

template

增强编辑#

在 Obsidian 中，可以通过“增强编辑”快捷键或安装 Trim Whitespace 插件快速删除多余空行。

https://github.com/obsidian-canzi/Enhanced-editing 好用，但是商店没法直接安装， 手动下载最新的压缩包，然后将其中文件(main.js, manifest.json) 解压到 库目录/.obsidian/plugins/ZH增强编辑 文件夹即可.

我的需求是这样的，虽然写得比较乱

使用obsidian，我的工作习惯是每天会在daily文件夹里面新建一个文件，比如今天是6越28日，会用6-28命名，然后当作流水账日志。有时候我会遇到一些要做的事情想记录，缺少一个统一的视图实时汇总我的todo 我的todo可能包含这些：比如一些标注了时间和日期的，算是紧急和时间确定的任务，然后有时候会写这周需要，这个月需要，以后有空等多种等级 有什么插件可以满足我“实时显示daily文件夹和private文件夹（有时候我觉得daily文件夹内东西太多，会将部分md文件的文字复制粘贴到总的年度日记里面。）内的todo？ 以及为我的需求设定一个书写的模板，我不想调用任何ai，所以尽量要求文字排版上能够统一，由于我的拖延症比较严重加上喜欢明日复明日，所以“有时候做””下周做“这种列表可能拉了一堆但一直不做，告诉我怎样进行人物的安排，让我知道优先做必要、紧急、重要的，一些不是很重要的可以留着我空闲的时候做
