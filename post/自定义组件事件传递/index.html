自定义组件事件传递

April 14, 2021 · map[avatar:images/avatar.jpg bio:Make milong famous again links:[map[name:GitHub url:https://github.com/milong26]] name:Milong]

微信小程序一个自定义组件Component，调用它的事件传递过程

Components#

新建component，其中有wxss、wxml、js、json四个文件

wxss和wxml#

定义组件的样式，和普通的页面一致

js文件#

properties#

（类似于data）存放数据

properties: {
    tabs:{
      type:Array,
      value:[]
    }
  },

methods#

在wxml中设置点击事件的话，调用的事件函数放在这里

  /**
   * 组件的方法列表
   */
  methods: {
    handleItemTap(e)
    {
      const {index}=e.currentTarget.dataset;
      // console.log(index);
      // 向调用这个组件的传递事件tabsItemChange和参数index
      this.triggerEvent("tabsItemChange",{index})
    }
  }

Page#

首先json文件中usingComponents需要引入它：

"usingComponents": {
    "Tabs":"../../components/tabs/Tabs"
  },

“Tabs"则作为这个组件的名字，在wxml中调用

<Tabs></Tabs>

需要传递参数tabs（写在js文件的data中）

<Tabs tabs="{{tabs}}" ></Tabs>

当从组件中传来事件响应时，需要监听事件，名字=bind+之前的传递事件名

<Tabs tabs="{{tabs}}"
bindtabsItemChange="handleTabsItemChange">
</Tabs>
