epoch和step什么关系？ 一个epoch的step就相当于data数量 除以 batch_size 大小

epoch：表示整个训练数据集被「完整地」训练了一遍。

step：指每次模型更新参数的一次操作，通常对应一次 forward + backward + optimizer.step()。

batch_size：每个 step 中喂给模型的样本数量。

怎么从一个链接下载内容 wget -P(一定要大写) 可以指定下载到本地的目录

ubuntu卸载一些软件，比如tencent meeting:sudo dpkg -P wemeet 卸载intellij idea：忘记当时怎么安装的了

  1. snap list | grep idea 如果是snap安装，会看到 sudo snap remove intellij-idea-community 就能卸载掉了 卸载默认的扫雷游戏 sudo apt remove gnome-mines sudo apt autoremove 其它的基本都这样

重启键盘，启动 IBus 输入法框架守护进程，-r 的意思是 restart，即“重启”。-d 的意思是 daemonize，即“以守护进程方式运行”。守护进程意味着这个程序会在后台运行，不会占用当前终端。如果不加 -d，它会在前台运行，并把日志输出到当前终端。-x 的意思是 enable XIM support。

ibus-daemon -r -d -x

给文件夹重命名（命令行） mv 旧文件夹名 新文件夹名

复制并重命名 cp -r 源文件 目标路径/新文件名。 -r 是为了复制目录及其内容，必须加上 -r（递归）选项。

检查空间使用情况 du -sh ~ 会显示总共的

du -sh ~/* 显示所有文件夹的（一层）

du -sh ~/* ~/.* 显示包含隐藏文件的

sync 没有输出了 lsof +D /media/qwe/xiake/collect_data | grep -v “bash|lsof” 检查这个文件夹有没有新增

共享文件夹#

文件夹右键有个，直接点击显示service not installed，点击install会安装Samba 不要允许访客访问

从 Ubuntu/Linux 系统中访问共享文件夹，你需要系统的 IP 地址或主机名。为此，打开系统设置(System Settings)_ -> Wi-Fi -> 获取 IP 地址(Get the IP address)”。

然后创建密码

sudo smbpasswd -a 指定的用户名 然后启用 sudo smbpasswd -e 指定的用户名

linux就搞定了 hostname -I 得到ip 去windows，资源管理栏//ip 就能打开，然后输入用户名和密码就ok
