为什么打开的终端默认是powershell，powershell是什么

PowerShell 是微软开发的一种命令行外壳和脚本语言，专为系统管理和自动化任务设计。它比传统的 CMD（命令提示符）功能更强大。

如何更改 VSCode 的默认终端？ 如果你想用其他终端（比如 CMD、Git Bash 或 WSL），可以这样做：

打开命令面板：Ctrl + Shift + P。

输入并选择：Terminal: Select Default Profile。

选择你想用的终端，比如 Command Prompt, Git Bash, WSL, 等。
