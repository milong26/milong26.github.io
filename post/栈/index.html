后入先出

ADT:

抽象数据类型Stack{
实例
	线性表，栈底，栈顶
操作
create()：创建一个空的堆栈
empty()：如果堆栈为空，则返回true，否则返回false
top()：返回栈顶元素
push(x)：向堆栈中添加元素x
pop()：删除栈顶元素
}

由左至右扫描表达式串，遇左括号，push

遇右括号，与栈顶左括号匹配，pop

右括号都处理完时，还有未匹配的左括号——表达式串处理完时，栈不空

举例

#include<iostream>
#include<string>
#include<stack>
const int max = 100;
using namespace std;
int main()
{
	char s[max];
	stack<char> st;
	while (cin >> s)//能多次输入，就跟之前那个扫雷似的
	{
		while (!st.empty())
			st.pop();//清空栈
		int len = strlen(s),flag=1;
		for (int i = 0; i < len; i++)
		{
			if (s[i] == '('||s[i]=='[')
				st.push(s[i]);
			else
			{
				if (st.empty())
				{
					flag = 0;
					break;
				}
				char t = st.top();
				st.pop();
				if (s[i] == ')' && t == '[')
				{
					flag = 0;
					break;
				}
				else if (s[i] == ']'&& t == '(')
				{
					flag = 0;
					break;
				}
			}
		}
		if (!st.empty()) flag = 0;
		if (flag == 1) cout << "yes" << endl;
		else cout << "no" << endl;
	}
    return 0;
}

缓冲轨后进先出，用堆栈保存车厢号

•考虑在出轨顺序，必须栈底大，栈顶小

依次检查入轨车厢编号

•如果≠出轨所需要的下一车厢，à缓冲轨

••依次检查缓冲轨，若新来的<栈顶，入栈

•如果=出轨所需要的下一车厢，->出轨

••缓冲轨中车厢可能满足出轨需要，检查缓冲轨栈顶车厢，如有可能，出栈，à出轨，不是一次，要反复做，直至栈中无满足出轨需要的车厢
