双拼

June 22, 2026 · map[avatar:images/avatar.jpg bio:Make milong famous again links:[map[name:GitHub url:https://github.com/milong26]] name:Milong]

又想学双拼了，现在才发现我的思维真是发散。

技术#

  1. 安装，打算使用小鹤双拼，搜索小鹤双拼windows 输入法，直接找修改注册表的方式简单一点
    1. Windows + R调出命令运行框，输入：regedit后确认打开注册表；
    2. 打开以下路径：HKEY_CURRENT_USER\Software\Microsoft\InputMethod\Settings\CHS
    3. 新建字符串值，字符串名称为：UserDefinedDoublePinyinScheme0，值为：小鹤双拼*2*^*iuvdjhcwfg^xmlnpbksqszxkrltvyovt
    4. 退出后进入微软拼音设置，验证小鹤双拼方案是否存在
  2. 当我还在练习的时候还是需要全拼的怎么办
    1. 参考https://blog.csdn.net/weixin_47193185/article/details/144597106
    2. 新建txt，内容文件后缀改为bat，保存ANSI。

@echo off

set Mainkey=HKCU\SOFTWARE\Microsoft\InputMethod\Settings\CHS

for /f %%i in (‘reg query %MainKey% /v “Enable Double Pinyin” ^| findstr /i “0x1”’) do (set flg=%%i)

if not defined flg ( reg add %MainKey% /v “Enable Double Pinyin” /t REG_DWORD /d 0x1 /f echo 已经切换到双拼 ) else ( reg add %MainKey% /v “Enable Double Pinyin” /t REG_DWORD /d 0x0 /f echo 已经切换到全拼 )

pause

   ```
  1. 随便原来的保存在哪里，然后创建一个快捷方式保存到桌面或者开始菜单文件夹里，右键快捷方式打开属性，点击快捷键，然后按下你希望的快捷键即可完成原生微软输入法的全拼和双拼的切换。我设置的是ctrl+shift+p。会弹出cmd窗口有点丑，但是可以接受。

学习历程#

当我想学一个新东西的时候，我需要做什么？

系统上学一个新东西的时候应该做什么？首先要判断自己的起点：我现在已经会什么、缺什么、距离目标还有多远；然后明确需求和目标，包括使用场景、掌握标准和完成时间。知道自己为什么学之后，再寻找合适的学习资料，优先找能建立整体认知的总览性内容，了解这个领域的结构、关键概念和学习路径，再进入入门教程学习基础操作。学习过程中要设计练习方式和反馈机制，通过实际应用发现问题，再调整学习计划，而不是只靠被动看资料。

对于我学习双拼这个需求，可以先分析现状：我现在可能只会全拼，习惯拼音输入，但不知道双拼方案和键位；目标可以设为“30天内能用双拼完成日常聊天输入，速度达到接近全拼水平”。然后搜索资料时，不应该一开始只搜“如何背双拼键位”，而应该先找“双拼输入法原理介绍”“双拼方案对比”“双拼学习路线”这类总览内容，先理解声母、韵母映射和不同方案的区别。确定方案后，再找“小鹤双拼入门教程”“双拼键位表”“双拼练习工具”等入门资料，按照记忆键位、练习音节、输入词语、真实使用的顺序训练。
