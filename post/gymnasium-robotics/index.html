Gymnasium-Robotics

March 10, 2025 · map[avatar:images/avatar.jpg bio:Make milong famous again links:[map[name:GitHub url:https://github.com/milong26]] name:Milong]

又来看新环境 Gymnasium-Robotics，看上了它提供的shadown hand

说明书

环境#

  1. linux服务器，cat /proc/version结果为Linux version 4.15.0-208-generic (buildd@lcy02-amd64-015) (gcc version 7.5.0 (Ubuntu 7.5.0-3ubuntu1~18.04)) #220-Ubuntu SMP Mon Mar 20 14:27:01 UTC 2023。
  2. 说明书写了支持Python 3.8, 3.9, 3.10 and 3.11 on Linux，选3.10

安装#

  1. conda create -n "gym" python==3.10
  2. conda activate gym
  3. 新版本的mujoco自动匹配mujoco-py了，就直接安装新版本mujoco pip install mujoco
  4. pip install gymnasium-robotics 最好加–default-timeout=100 或者更换镜像源，不然会报超时错误
  5. print (gym.version) 1.1.0
  6. print (gymnasium_robotics.version) 1.3.1

使用#

  1. 怎么控制shadow hand的，虽然知道大概是强化学习。
  2. 四个任务(环境)
    1. HandReach-v2：ShadowHand必须用拇指和选定的手指到达手掌上方的目标位置，直到它们相遇。
    2. HandManipulateBlock-v1：ShadowHand必须操纵一个块，直到它达到所需的目标位置和旋转。
    3. HandManipulateEgg-v1：ShadowHand必须操纵一个蛋，直到它达到所需的目标位置和旋转。
    4. HandManipulatePen-v1：ShadowHand必须操纵一支笔，直到它达到所需的目标位置和旋转。
  3. 不需要运行代码，只要了解某一个任务
    1. 怎么控制hand dof？pos？angle？
    2. 怎么判断是否完成任务？ 问题1，2其实就是强化学习的事儿
    3. 有什么数据集可以拿过来用 ？
