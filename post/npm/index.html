淘宝镜像过期 提示：request to https://registry.npm.taobao.org/xxx failed, reason: certificate has expired 三、解决方案

1、查看当前的npm镜像设置：npm config list

2、清空缓存：npm cache clean –force

3、然后修改镜像即可：npm config set registry https://registry.npmmirror.com 一般国内用户还是建议使用淘宝镜像，所以推荐还是设置成用淘宝镜像 4、再次运行： npm config list，查看 registry 已经被更改为默认的 npm 公共镜像地址。
