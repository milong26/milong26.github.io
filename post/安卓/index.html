我之前没有安装过adb嘛？ https://developer.android.google.cn/tools/releases/platform-tools?hl=zh-cn 点击下载适用于 Windows 的 SDK Platform-Tools 提取到application/adb这个文件夹里面 为了以后方便，配置个环境变量 选择Path，将ADB的安装路径（即安装目录D:\application\adb\platform-tools）添加至path中即可。 cmd中 adb version有结果就行

然后尝试关闭振动，嫌烦 adb shell settings put system haptic_feedback_enabled 0 没用，→ 关闭的是 Android 系统级“触摸触感反馈”开关，其实我系统里面已经关掉了 然后看是不是obsidian的问题 adb shell pm list packages | findstr obsidian 检查包名得到package:md.obsidian adb shell appops set md.obsidian VIBRATE ignore 终于关掉了，是调用马达的
