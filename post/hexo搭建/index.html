hexo搭建个人blog，相关配置信息

预备#

  • node
  • git
  • 一个名为1021048313.github.io的仓库

开始#

  1. 新建一个文件夹存放网站文档并进入该文件夹，git bash here

     npm i hexo-cli -g

  2. 将D:\conf\node\node-v14.16.0-win-x64\node-global\node_modules\hexo-cli\bin加入Path变量

    hexo -v
hexo init

  3. hexo g生成静态网页

    hexo s打开本地服务器

    localhost:4000

  4. 配置

    _config.yml：修改最后的deploy

    deploy:
  type: git
  repository: git@github.com:1021048313/1021048313.github.io
  branch: main

    安装git插件

    npm install hexo-deployer-git –save

  5. hexo new post “article title"新建一个post，在/source/_posts中可以找到md文件

    hexo d就行

  6. 绑定域名

    ping 1021048313.gitub.io找到ip

    添加A记录 www 、@，值为刚刚得到的ip（一个就行）

    settings-pages里找到填域名的地方
