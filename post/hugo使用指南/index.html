需求清单

  • 能联动pages（pages只支持静态吧）
  • 可以有热力图
  • 可以显示公式
  • 有标签和分类
  • 可以obsidian

开搞 参考别人的博客 都是windows环境

环境配置#

hugo#

安装#

下载v0.156.0release搜索到有windows版本的，最好直接安装extended版本：hugo_extended_withdeploy_windows命名的文件下载 设置环境变量，Path中添加hugo解压后的文件夹（可以访问到hugo.exe的那个路径） hugo version检查有结果了就行

教程

使用前的配置#

在期望存放的位置

hugo new site guance

出现Congratulations! Your new Hugo project was created in表示成功 进入文件夹，必须先找到一个主题，用PaperMod吧

git init  
git submodule add https://github.com/adityatelange/hugo-PaperMod.git themes/PaperMod

然后修改 hugo.toml：

baseURL = "https://example.com/"  
languageCode = "en-us"  
title = "My Notes"  
theme = "PaperMod"

重点是改theme

然后执行hugo就能转到public

hugo server -D

可以预览

hugo

可以生成

将网站文件夹转换为 Git ，要去public文件夹然后git init

将 Git 本地库关联至远程库。

已有的md文件复制粘贴到blog/content/posts/ 创建模板（借助obsidian）

title: "{{title}}"
